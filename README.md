# Todo App

Веб-приложение для управления задачами с регистрацией и авторизацией пользователей.

## Технологии

- Go 1.26.2
- SQLite (database/sql + go-sqlite3)
- bcrypt для хеширования паролей
- Стандартная библиотека net/http
- Vanilla JS на фронтенде

## Возможности

- Регистрация и вход в аккаунт
- Задачи привязаны к пользователю, каждый видит только свои
- Добавление задач
- Отметка задачи как выполненной
- Удаление задач
- Статистика: всего, выполнено, осталось

## Как запустить

Убедись что установлен Go и gcc.

Клонируй репозиторий:

git clone https://github.com/SurP123/todo.git
cd todo

Установи зависимости:

go mod tidy

Запусти сервер:

go run .

Открой в браузере: http://localhost:8181
